# Forgotten Tech Map

Forgotten Tech Map is a small Kotlin application that catalogs obsolete programming languages, operating systems and file formats.

The project demonstrates a simple repository architecture with search, filtering and timeline generation.

## Categories

- Programming Languages
- Operating Systems
- File Formats

## Features

- Browse archived technologies
- Search by name
- Filter by category
- Display release years
- Print statistics
